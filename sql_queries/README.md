# Healthcare Analytics SQL & Tableau Project

## Project Overview
This project analyzes synthetic healthcare operational performance across hospitals and boroughs using PostgreSQL and Tableau.

## Tools Used
- PostgreSQL
- SQL
- Tableau Public
- Excel

## Key Metrics
- Readmission Rates
- Treatment Costs
- Patient Satisfaction
- Length of Stay
- Admission Trends

## Key Insights
- Heart Failure generated the highest average treatment costs.
- Certain boroughs showed higher readmission rates.
- Patient satisfaction varied across hospitals.
